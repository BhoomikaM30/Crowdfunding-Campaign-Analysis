# Crowdfunding-Campaign-Analysis
Data analysis project exploring crowdfunding campaign performance using SQL, Microsoft Excel, Tableau and Power BI.
This project was completed as part of my Data Analyst Internship at AI Variant.

The objective of the project was to analyze crowdfunding campaign data using SQL, Microsoft Excel, Tableau and Power BI to identify trends, campaign performance and key business insights.

More project details, dashboards and analysis will be added as the repository is updated.
